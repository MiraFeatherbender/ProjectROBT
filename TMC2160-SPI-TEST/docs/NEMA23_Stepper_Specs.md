# NEMA23 Stepper Motor Specifications

| Parameter               | Value                        |
|-------------------------|------------------------------|
| Phase Number            | Bipolar                      |
| Driving Voltage         | 24–48V DC (Recommended)      |
| Coil Current            | DC 2.8A / phase              |
| Resistance (20°C)       | 1.1 ±10% Ω / phase           |
| Inductance              | 3.0 ±20% mH / phase          |
| Step Angle              | 1.8° ±5%                     |
| Holding Torque          | ≥ 12.6 Kg-cm (175 Oz.in)     |
| Detent Torque           | 400 g-cm REF.                |
| Rotor Inertia           | 300 g·cm²                    |
| Weight                  | 0.75 Kg REF.                 |
| MAX Slewing PPS         | 5000 PPS                     |
| MAX Starting PPS        | 2500 PPS                     |
| Rotate Direction        | ABCDA CW                     |
| Insulation Resistance   | ≥100 MΩ (DC 500V)            |
| Insulation Class        | Class B                      |
| HI POT                  | AC600V / 1mA / 1S            |

---

**Notes:**

- PPS = Pulses Per Second
- REF. = Reference value
- All values are per phase unless otherwise specified.
